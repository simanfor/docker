Docker Cheat Sheet

Guía rápida de comandos esenciales para Ubuntu (Linux) y Windows (PowerShell/CMD)

1. Gestión de Contenedores (Ciclo de Vida)

Comando Descripción SO
docker ps
 Listar contenedores activos. Ambos
docker ps -a
 Listar todos los contenedores (activos e inactivos). Ambos
docker run -d --name mi_contenedor nombre_imagen
 Crear y arrancar un contenedor en segundo plano (detached). Ambos
docker start mi_contenedor
 Iniciar un contenedor detenido.
